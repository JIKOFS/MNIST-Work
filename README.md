# 机器学习大作业 - 实验报告代码

这是《机器学习工程实践》大作业代码仓库。

这个项目包含了两个主要部分，分别对应课程作业中要求的 KNN 和 FNN 任务。所有的代码用 Python 实现，并且加了详细的中文注释，方便回顾。

## 目录结构

整个项目的结构是这样的：

```text
ML_Assignment/
├── 01_KNN_MNIST/          # 任务一：用 KNN 算法识别手写数字
│   └── run_knn.py         # 这个脚本会下载数据，然后用我手写的 KNN 跑分类
├── 02_FNN_MNIST/          # 任务二：用神经网络 (FNN) 识别数字
│   ├── model.py           # 定义神经网络长什么样 (3层结构)
│   ├── train.py           # 训练模型用的脚本 (跑完会生成一个模型文件)
│   ├── gui_app.py         # 一个可以直接在屏幕上写数字的识别程序
│   └── models/            # 训练好的模型会存在这里
├── data/                  # 数据集目录 (已包含 MNIST 数据)
└── requirements.txt       # 这是一个清单，列出了需要安装的 Python 库
```

---

## 配置环境

建议使用 Python 3.8 或以上版本。

在项目根目录下，打开命令行（终端），运行：

```bash
pip install -r requirements.txt
```

## 运行代码

### 1. 任务一：KNN 手写数字分类
这个任务是为了理解 KNN 算法的原理。没有调用现成的库，而是用 NumPy 矩阵运算实现了 KNN。

**运行方法：**
```bash
cd 01_KNN_MNIST
python run_knn.py
```
*注：默认使用全部 10000 张测试图进行验证。数据集已下载好，代码会直接加载 `../data` 目录下的文件。*
### 2. 任务二：神经网络 (FNN) 识别
这个任务是用 PyTorch 搭建一个简单的全连接神经网络，用来识别 MNIST 数据集，并支持手写拍照识别。

**第一步：训练模型**
```bash
cd 02_FNN_MNIST
python train.py
```
跑完后，会看到准确率大概能到 97% 左右，并且会生成一个 `models/mnist_fnn.pth` 文件。

**第二步：玩一下手写识别 (GUI)**
这是我做的一个小程序，可以在屏幕上直接写字，AI 会实时猜你写的是什么。
```bash
python gui_app.py
```
*(记得要先运行上面的训练脚本，不然会报错说找不到模型)*
